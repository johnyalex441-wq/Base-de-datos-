# Entrega de primera administracion de BD-
Trabajo de base de datos en MySQL del equipo N°3

La base de datos trata de una tienda de accesorios para PC la cual contiene 
5 tablas principales con 25 datos cada una. La cual muestra como guarda los datos
de los productos e informa sobre el control de empleados, clientes asociados, las ventas 
realizadas y distribuidores con los que trabajan.
